# Build Local AI-Powered Chatbots with LangChain & Ollama

Welcome to the updated 2026 edition of this beginner-friendly project from **HERE AND NOW AI – Artificial Intelligence Research Institute**. You will learn how to build private, local, and multimodal **AI-powered chatbots** using **LangChain**, **Ollama**, and **Gradio**.

---

## 🚀 What You'll Learn

In this project, you'll learn:

- **Local LLM Development**: Run models like `hereandnow-ai` and `qwen3-vl` on your own machine.
- **LangChain Integration**: Use the modern LangChain Ollama stack for AI logic.
- **RAG (Retrieval-Augmented Generation)**: Chat with your own PDFs, Websites, and Text files.
- **Multimodal AI**: Build bots that can "see" and describe images locally.
- **Persona Management**: Centralize your AI's personality (using the **Caramel AI** persona).

---

## 🧠 Who This Is For

- Students learning AI for the first time  
- Educators teaching beginner AI courses  
- Professionals concerned about data privacy (local-only stack)
- Researchers exploring GenAI tools without API costs

---

## 🛠 Tech Stack

- **Python 3.11+**
- **Ollama** (Local Model Hosting)
- **LangChain** (AI Framework)
- **Gradio 6.0** (User Interface)
- **FAISS** (Local Vector Database)

---

## 📁 Project Structure

- `config.py` - Central configuration for models and personality.
- `project_1_ai_chatbot_without_memory/` - Simple text chatbot.
- `project_2_ai_chatbot_with_memory/` - Chatbot with conversational history.
- `project_3_ai_ragbot_with_text/` - Chat with local text files.
- `project_4_ragbot_with_web/` - Chat with any website content.
- `project_5_ai_ragbot_with_pdf/` - Simple PDF-based RAG.
- `project_7_ai_ragbot_with_vector_db/` - Advanced RAG using local vector storage.
- `project_8_ai_ragbot_with_image/` - Local multimodal vision assistant.

---

## ✅ Prerequisites

- [Ollama](https://ollama.com/) installed and running.
- Local Models: `hereandnow-ai:latest`, `qwen3-vl:2b`, and `nomic-embed-text:latest`.
- Python 3.11 environment (see `requirements.txt`).
